# Game Caro

## Server

- Server sử dụng Nodejs, các thư viện express, socket.io hỗ trợ kết nối với client.
- Server xử lý các logic của game, lưu trữ tạm các thông tin cơ bản.
- Run server:

```sh
cd Server
npm install
node app.js
```

## Client

- Client sử dụng React, thư viện socket.io hỗ trợ kết nối.
- Run client:

```sh
cd Client
npm install
npm start
```
